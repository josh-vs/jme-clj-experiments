# jme-clj-experiments

Experimenting with a 3D game in JMonkeyEngine in Clojure - trying out shaders and physics

For this project I'm using the Calva extension for VSCode.

You can try it out by starting a Leinengen REPL and evaluating everything in `src/jme_clj_experiments/modelmovement.clj` from the top until `(jme/start app)` which will start a window with the game running.
